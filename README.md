## Despliegue 📦

> El proyecto se ejecuta sobre el puerto process.env.PORT o 8080

* Instalacion de depedencias
```bash
npm i 
```
* Ejecucion del proyecto handlebars (se ejecuta en puerto 8080)
```bash
npm run start
```
